Gold Coast Surf Forecast

Web app

Gold Coast Surf Forecast is a web application built with Python and Flask to offer real-time updates on surf conditions across all the prominent spots along the Gold Coast of Australia. The back-end of the application leverages the power of SQL and SQLAlchemy to manage the database system. On the front-end, the interface of Gold Coast Surf Forecast was designed using HTML and CSS, combined with Jinja for template rendering.

With Gold Coast Surf Forecast users can create personalized accounts and save their favorite surf spots to a dedicated favorites page, allowing quick and easy access to detailed surf conditions specific to their preferences. The application is designed to provide real-time updates on surf conditions, including wave height, wind speed, tide times, and other critical metrics essential for surfers.

Stocks

Web app

Stocks is a web application where users can create a personalized account and manage a portfolio of stocks. I created Stocks as a project while completing Harvard's CS50 course. This application is built using Python and Flask. The back-end of the application uses SQL and SQLAlchemy, allowing the management of a database system for each user account and their portfolio of stocks. On the front-end, the application's interface was designed using HTML and CSS, combined with Jinja for template rendering.
